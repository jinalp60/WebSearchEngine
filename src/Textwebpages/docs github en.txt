Connecting to GitHub with SSH You can connect to GitHub using the Secure Shell Protocol (SSH), which provides a secure channel over an unsecured network. Creating and managing repositories You can create a repository on GitHub to store and collaborate on your project's files, then manage the repository's name and location. Popular About pull requests Pull requests let you tell others about changes you've pushed to a branch in a repository on GitHub. Once a pull request is opened, you can discuss and review the potential changes with collaborators and add follow-up commits before your changes are merged into the base branch. Authentication Keep your account and data secure with features like two-factor authentication, SSH, and commit signature verification.
